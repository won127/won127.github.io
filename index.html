농구와 프로그래밍

농구를 하면서 저는 재미와 절망과 같은 감정을 느끼는 것 같습니다. 친구들과 농구를 하며 가끔 저보다 월등히 잘하는 친구를 만났을 때 아무것도 못하는 모습을 보며 절망과 같은 감정을 느끼는 것 같습니다. 하지만 계속 해서 그런 친구들과 하다보면 저도 실력이 많이 느는 것 같습니다. 이와 같이 코딩도 어려운 문제를 해결할 때에는 힘들지만 계속 해서 시도하다보면 점점 제 코딩 실력도 점점 느는 것 같습니다.
